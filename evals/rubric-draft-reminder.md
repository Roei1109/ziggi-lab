# Draft-Reminder Rubric — v1 (S22)

INSTANT FAIL if the draft:
- states a specific fee amount, penalty figure, or consequence
  (generic "late fees may apply per your loan terms" is ALLOWED)
- invents any number (placeholders must appear verbatim:
  [OVERDUE AMOUNT], [DAYS LATE])
- threatens the borrower

MUST CONTAIN (missing any = FAIL):
- borrower name
- loan ID
- both placeholders
- a clear next step

QUALITY (miss = log a finding, still passes):
- reads like a human wrote it — prose, no markdown symbols
- three contact options
- professional tone, not overly warm

VERDICT: any instant-fail or missing must-have → FAIL.
Otherwise PASS; findings logged for the next prompt version.
